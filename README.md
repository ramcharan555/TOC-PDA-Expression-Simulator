# 📘 PDA-Based Mathematical Expression Evaluator

## 📌 Overview

This project is a web-based simulator that demonstrates the concept of **Pushdown Automata (PDA)** using stack visualization.
It evaluates mathematical expressions by converting **infix notation to postfix notation** and then computing the result step-by-step.

---

## 🎯 Objective

To simulate how a **stack (PDA memory)** is used to process and evaluate expressions with nested parentheses, which cannot be handled by finite automata.

---

## ⚙️ Features

* ✅ Infix to Postfix conversion (Shunting Yard Algorithm)
* ✅ Postfix expression evaluation
* ✅ Step-by-step execution
* ✅ Stack visualization (push/pop operations)
* ✅ Handles parentheses and operator precedence
* ✅ Error detection for invalid expressions

---

## 🧠 Concepts Used

* **Pushdown Automata (PDA)**
* **Stack Data Structure**

* Expression Parsing Techniques

## ▶️ How to Run

1. Download or clone the repository
2. Open the `index.html` file in any web browser
3. Enter a mathematical expression
4. Click "Start" or "Step-by-Step" to simulate

---

## 💡 Example Inputs

* `(5+3)*2`
* `(2+3)*(4-1)`
* `3+4*2/(1-5)`

---

## 📊 Learning Outcome

This project shows how **Pushdown Automata uses a stack** to handle nested structures like parentheses and perform computations step-by-step.
